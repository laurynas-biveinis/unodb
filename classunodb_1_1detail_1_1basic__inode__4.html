Detailed Description

template<class ArtPolicy>
class unodb::detail::basic_inode_4< ArtPolicy >

An internal node with up to 4 children.

There is an array of (4) bytes for the keys and keys are maintained in lexicographic order. There is a corresponding array of (4) child pointers. Each key position is an index into the corresponding child pointer position, so the child pointers are also dense.
