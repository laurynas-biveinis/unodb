Detailed Description

template<class Key, class Header>
class unodb::detail::basic_leaf< Key, Header >

The basic_leaf handles most of the behavior of a leaf in the index.

It is specialized for OLC which includes additional information in the header. The leaf contains a copy of the key and a copy of the value.
