Detailed Description

template<class Header>
class unodb::detail::basic_node_ptr< Header >

basic_node_ptr is a tagged pointer (the tag is the node type).

You have to know statically the target type, then call node_ptr_var.ptr<target_type *>.ptr() to get target_type.
