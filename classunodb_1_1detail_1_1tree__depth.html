Detailed Description

template<typename ArtKey>
class unodb::detail::tree_depth< ArtKey >

Number of key bytes consumed along some path in the tree.

In general, the unodb::detail::key_prefix consumes up to some fixed number of bytes and one byte is consumed based on the key_byte for each node as we descend along that byte value to some child index.
