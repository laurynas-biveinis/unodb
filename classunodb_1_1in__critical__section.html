Detailed Description

template<typename T>
class unodb::in_critical_section< T >

Gloss for the atomic semantics used to guard loads and stores.

Wraps the protected data fields. The loads and stores become relaxed atomic operations as required by the optimistic lock memory model. The instances are non-moveable and non-copy-constructable but the assignments both from the wrapped values and plain value type are supported.

To support reusing the same code for single-threaded context too, there is a no-op counterpart: unodb::in_fake_critical_section, enabling templatizing on the wrapper type and passing either class as needed.

Implements the required set of transparent operations, extend as necessary.
