Detailed Description

template<typename T>
class unodb::in_fake_critical_section< T >

Provide access to T with unodb::in_critical_section<T>-like interface, except that loads and stores are direct instead of relaxed atomic.

It enables having a common templatized implementation of single-threaded and OLC node algorithms.
