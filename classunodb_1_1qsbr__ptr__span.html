Detailed Description

template<typename T>
class unodb::qsbr_ptr_span< T >

An std::span, but with unodb::qsbr_ptr instead of a raw pointer, that is, a span over QSBR-managed data.

Crashes debug builds if a thread goes through a quiescent state while having an active span. Meets C++ contiguous range requirements.
