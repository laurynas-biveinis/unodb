Detailed Description

template<typename ArtKey, template< class > class CriticalSectionPolicy>
union unodb::detail::key_prefix< ArtKey, CriticalSectionPolicy >

The unodb::detail::key_prefix is a sequence of zero or more bytes for a given node that are a common prefix shared by all children of that node and supports prefix compression in the index.
